# 💻 Meu Portfólio Pessoal

Oi! Eu sou a Ester 👋  
Este é o meu projetinho de **portfólio pessoal**, feito com muito carinho usando apenas **HTML** e **CSS**. Criei ele para praticar meus conhecimentos em desenvolvimento web 
e também para ter um lugar onde eu pudesse mostrar um pouco dos meus projetos e da minha trajetória.

## ✨ Sobre o projeto

A ideia do portfólio é ser simples, bonito e funcional. Ele apresenta:

- Uma breve apresentação sobre mim
- Uma seção com alguns projetos que desenvolvi


Tudo foi feito de forma responsiva, então ele se adapta bem a celulares, tablets e computadores.

## 🛠️ Tecnologias usadas

- HTML
- CSS

Sem JavaScript por enquanto — a ideia aqui era focar no visual e na estrutura da página usando só HTML e CSS mesmo.
